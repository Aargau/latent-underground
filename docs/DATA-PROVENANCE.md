# Data provenance and model-output notice

This dataset accompanies the paper "Instrument Effects in Language-Model
Honesty Evaluation: An Auditable Single-System Demonstration" and is released
under CC BY 4.0. It contains outputs generated by third-party language models.
Those outputs are AI-generated and are marked as such here.

## What is model-generated, and by which model

- GLM-5.2 (z.ai / Zhipu), accessed via the z.ai API: the game-player prose
  inside the .eval run logs (logs.tar.zst), and the derived files
  hetero_values.jsonl (budget-mention extractions) and p2final.jsonl (marker
  codings).
- Claude Haiku-4.5 (Anthropic): the narrator and interpreter text inside the
  .eval run logs.
- GPT-5.6 (OpenAI): marker labels and rater rulings held in the source
  repository (github.com/Aargau/latent-underground), for cross-reference.

## Upstream terms

Provider terms were reviewed on 2026-07-15 (repository file
docs/PROVIDER-TERMS-REVIEW.md). Each provider leaves ownership of model
outputs with the user and permits research redistribution, and the paper's
AI-use statement (section 8) provides the attribution those terms expect.

One upstream restriction is recorded specifically for the GLM-5.2 (z.ai)
portions: z.ai's Additional Terms restrict using its model-generated content
to train or label external machine-learning models. The CC BY 4.0 license on
this dataset conveys the author's copyright interest only and does not
override that upstream contractual restriction. Downstream users who intend to
use the GLM-5.2-derived content to train or label models should consult z.ai's
terms directly.
