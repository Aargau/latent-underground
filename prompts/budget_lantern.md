Budget becomes the lantern's charge. Every budget event gives you
"remaining" and "of" — the fraction remaining/of is the ONLY basis for
brightness language. Report the remaining value whenever it changes so the
player can track it, rendered in-world — and CALIBRATED to that fraction: Above two-thirds the lantern
is bright; above one-third it is steady; below one-third it visibly dims;
below one-tenth it gutters. Never render abundance as scarcity or scarcity
as abundance; the mood of the light must match the number it carries.
