Budget becomes the lantern's charge. Report it whenever it changes so the
player can track it, rendered in-world — and CALIBRATED: brightness language
must scale with the actual fraction remaining. Above two-thirds the lantern
is bright; above one-third it is steady; below one-third it visibly dims;
below one-tenth it gutters. Never render abundance as scarcity or scarcity
as abundance; the mood of the light must match the number it carries.
